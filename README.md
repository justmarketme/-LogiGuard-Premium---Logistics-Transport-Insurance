## ğŸ“¦ Getting Started

### Prerequisites

- A modern web browser.
- A valid Google Gemini API Key.

### Environment Setup

The application expects an API key to be available in the execution environment:
`process.env.API_KEY`

### Local Development

Since this project uses ES6 modules directly in the browser via import maps:
1. Clone the repository.
2. Serve the root directory using a local web server (e.g., `npx serve .` or Live Server).
3. Open `index.html` in your browser.
